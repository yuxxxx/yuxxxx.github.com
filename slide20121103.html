CSSアニメーション

概要

  • 現在W3Cのワーキングドラフト

  • キーフレームを用いたアニメーション

  • アニメーションする時間や繰り返し回数、繰り返し方も設定可能

  • ベンダープレフィックスが少々厄介

CSSアニメーション

よくある光景

@keyframes swing {
  0% { transform:rotate(-10deg); }
  50% { transform:rotate(10.3deg); }
  100% { transform:rotate(-10deg); }
}
@-moz-keyframes swing {
  0% { -moz-transform:rotate(-10deg); }
  50% { -moz-transform:rotate(10.3deg); }
  100% { -moz-transform:rotate(-10deg); }
}
@-webkit-keyframes swing {
  0% { -webkit-transform:rotate(-10deg); }
  50% { -webkit-transform:rotate(10.3deg); }
  100% { -webkit-transform:rotate(-10deg); }
}
 .bone {
  animation: swing 1s ease-in-out 0s infinite normal;
  -moz-animation: swing 1s ease-in-out 0s infinite normal;
  -webkit-animation: swing 1s ease-in-out 0s infinite normal;
}

今後はベンダープレフィックスを省略します。

CSS Transitions

概要

  • 現在W3Cのワーキングドラフト

  • アニメーションというよりはスムーズな遷移(Transition)

  • プロパティの数値を時間をかけてゆるやかに変更させる

CSS Transions

実例

このスライドのページめくり(一部抜粋)

.csstransitions.csstransforms .deck-container > .slide {
  transition: transform 500ms ease-in-out;
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .slide .slide {
  transition: transform 500ms ease-in-out, opacity 500ms ease-in-out;
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .slide .deck-next, .csstransitions.csstransforms .deck-container:not(.deck-menu) > .slide .deck-after {
  visibility: visible;
  transform: translate3d(200%, 0, 0);
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-previous {
  transform: translate3d(-200%, 0, 0);
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-before {
  transform: translate3d(-400%, 0, 0);
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-next {
  transform: translate3d(200%, 0, 0);
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-after {
  transform: translate3d(400%, 0, 0);
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-before .slide, .csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-previous .slide {
  visibility: visible;
}
.csstransitions.csstransforms .deck-container:not(.deck-menu) > .deck-child-current {
  transform: none;
}

Canvas

概要

  • HTML5で新登場したタグ

  • JavaScriptを使って静止画を描画できる

  • requestAnimationFrameなどを使い、パラパラマンガの要領でアニメーション

  • ライブラリを使えば多少幸せになれる

  • 環境があれば３DCGも描画できる

Canvas

実例

http://jsdo.it/yuu_hara/iLbt

やってること
  1. Canvasをクリアしたあと
  2. 全体を黄色く塗りつぶし、
  3. 上側を白く塗りつぶしたあと
  4. 境目あたりに黒い線をいくつか引く

JavaScriptでエレメントを操作

  • requestAnimationFrameなどを使い、ごく短い時間で連続してスタイルを書き換える

  • アニメーションさせる原理はさっきとあんまり変わりない

  • ライブラリ無しでやるのはあんまり考えたくない
